# FastAPI Course Management API

A backend REST API built with **FastAPI** for managing courses and users.  
This project focuses on authentication, authorization, and clean API design using modern Python tools.
---
## Features
- User registration
- User login with JWT authentication
- Secure password hashing
- Protected API routes
- Course CRUD operations (Create, Read, Update, Delete)
- Dependency injection
- Environment-based configuration
---
## Tech Stack
- FastAPI
- SQLAlchemy
- PostgreSQL
- JWT Authentication
- Pydantic
- Python 3
